# 🏫 座位編排器 Pro 完整使用手冊

這是一款專為教育工作者設計的智慧化座位管理工具。透過直觀的介面與強大的隨機演算法，幫助您在幾秒鐘內完成符合教育需求的座位安排。

---

## 🚀 核心功能介紹

### 1. 教室佈局規劃
* **預設網格**：標準 $6 \times 6$ 共 36 個位置。
* **自訂座位類型**：點擊網格中的任何位置可開啟選單：
    * **一般座位**：參與隨機編排。
    * **👔 設為老師桌**：標記為導師辦公位置（不排學生）。
    * **🚫 設為禁坐區**：將該格子設為空置，可用於走道規劃或損壞課桌。
    * **📌 固定學生**：直接從名單中挑選學生鎖定在該位置，智慧編排時不會搬動他。

### 2. 學生名單與屬性
* **名單輸入**：在左側文字框內輸入姓名，每行一位。
* **★ 優先前排**：
    * 輸入名單後，下方會出現學生姓名標籤。
    * 點擊標籤可標記為「**優先（★）**」。
    * 系統會盡力將帶有星號的學生分配在**第一、二排**的位置（視力或注意力輔助）。

### 3. 避開規則 (黑名單)
* 為了防止特定同學相處時產生干擾，可設定「避開規則」。
* 輸入兩位學生的姓名並點擊「新增規則」。
* 演算法會確保這兩位學生**不會出現在彼此相鄰（上下左右）**的位置。

---

## 🛠 操作流程

1.  **輸入名單**：將班級名單貼入左側區域。
2.  **規劃環境**：點擊右側網格，將特定位置設為導師位或空位。
3.  **標記需求**：標記需要前排的學生，並設定需要分開坐的名單。
4.  **智慧編排**：點擊 **[🎲 開始智慧隨機編排]**。
    > **提示**：如果規則過於複雜（例如過多黑名單或太多優先位），系統可能會提示無法完成，此時請稍微放寬規則。
5.  **下載結果**：點擊右上角的 **[📥 下載座位表圖片]**，儲存 PNG 圖檔。

---

## 💡 小技巧

* **固定同學**：您可以利用「固定」功能，先將需要特別關心的同學或幹部排在特定的位置，再讓剩下的同學隨機分配。
* **自由拖動**：編排後若不滿意，可以隨時自由拖動更變同學的位置，若拖動的座位已經有人則會交換兩人位置。（亦可點擊重新分配）
* **下載前檢查**：下載圖片前，建議先關閉所有彈出的設定選單，以確保座位表內容清晰無遮擋。
