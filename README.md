# 🏫 座位編排器 Pro 官方使用手冊
  
<a href="https://707-tool-qpjh.vercel.app/cePqnw" target="_blank">點擊連結網站</a>

這是一款專為教育現場量身打造的**智慧化教室管理工具**。透過整合「隨機演算法」、「衝突避開機制」與「直覺式拖放介面」，  
本工具將原本繁瑣的換座位任務，轉化為幾秒鐘內的直覺操作。

---

## 💎 系統核心亮點

* **智慧權重演算**：自動優先處理「★ 需求學生」與「避開規則」，並進行千次模擬以求最優解。
* **📌 精準固定系統**：鎖定特定學生不參與隨機移動，適用於幹部或特殊需求學生佈陣。
* **自由拖放交換 (Drag & Drop)**：業界領先的實體感操作，編排後仍可隨時手動微調位置。
* **視覺化佈局編輯**：直覺式點擊選單，自由切換老師桌、禁坐區與一般座位。
* **一鍵導出**：整合 `html2canvas` 技術，產出包含圖例與黑板標示的專業 PNG 座位表。

---

## 🛠 功能操作全解析

### 一、 名單管理與權重標記
1.  **輸入名單**：在左側「名單管理」區貼入姓名。
    * *註：每行一位姓名，系統會自動識別總人數。*
2.  **設定優先權 (★)**：
    * 輸入名單後，下方會自動生成姓名標籤。
    * **操作**：點擊標籤使其變為黃色並出現「★」。
    * **邏輯**：演算法會強制將標記學生填入**前兩排**（1-12 號位），滿足視力輔助需求。

### 二、 戰略佈置：固定座位 (Fix Seat)
這是本系統最核心的「預佈陣」功能，允許老師在隨機編排前先手動排定特定位置：
* **操作流程**：
    1.  點擊右側網格中任何一個格子開啟設定選單。
    2.  向下滾動選單找到目標學生姓名。
    3.  **點擊姓名**後，該位置會立即變為 **「📌 固定態」**（橘色邊框與釘子圖示）。
* **功能邏輯**：一旦固定，該生便**不參與隨機移動**。按下隨機按鈕時，他會固定在原位，而其他學生則繞著他進行分配。

### 三、 環境自訂：教室佈局
點擊格子可切換物理屬性，以符合真實教室情況：
* **👔 設為老師桌**：深色區塊。標記為導師位置，演算法不會排入學生。
* **🚫 設為禁坐區**：斜紋區塊。標記為空課桌，演算法不會排入學生。

### 四、 社交距離：避開規則 (黑名單)
* **操作**：在「避開規則」欄位分別輸入 A 同學與 B 同學，點擊「➕ 新增規則」。
* **邏輯**：系統會確保這兩位學生在結果中，**絕對不會出現在彼此的「上下左右」相鄰位置**。
* **提示**：若規則過多導致無解，系統會提示「規則衝突」，此時請刪除部分規則。

### 五、 進階微調：手動拖放系統
編排完成後，若想進行最後微調：
1.  **滑鼠拖曳**：滑鼠左鍵「按住」格子內的學生姓名。
2.  **原地交換**：拖至目標格子放開，兩位學生的**姓名、優先權、固定狀態**會同步交換。
3.  **無縫銜接**：操作具備半透明反饋，手感順暢。
